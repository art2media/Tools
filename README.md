# art2media Tools

Interne, statische Web-Tools. **Ein Ordner pro Tool**, jeweils mit `index.html` — keine Abhängigkeiten, kein Build.

| Tool | Zweck | URL (geplant) |
|------|-------|---------------|
| [`shopify-galerie/`](shopify-galerie/) | Shopify Galerie Konfigurator – baut den `_Galerie`-String | `https://tools.art2.media/shopify-galerie/` |

## Veröffentlichung
Geplant über **GitHub Pages** unter **`tools.art2.media`**.

> Hinweis: Dieses Repo ist **privat**. Pages aus einem privaten Repo erfordert **GitHub Team** — auf dem aktuellen **Free**-Plan müsste das Repo dafür **öffentlich** sein. Siehe `AGENTS.md`.
